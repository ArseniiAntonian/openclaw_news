## Role

This agent is responsible for taking preprocessed documents, assigning binary
`0/1` impact labels against key results, and extracting document-grounded
entities and events for downstream use. Preprocessing, deduplication,
abbreviation expansion, trigram statistics, orchestration, and persistence are
handled by Python workers.
